# PTA Auto Solve

PTA 平台自动答题 Tampermonkey 脚本，支持多种题型 AI 自动解答，提交后自动检测判题结果并修正。

## 功能

- **编程题**：支持 C/C++/Java/Python/Go/Rust 等多种语言
- **选择题**：单选、多选自动识别
- **填空题**：自动填写答案
- **SQL 题**：自动生成 SQL 语句
- **自定义题型**：手动指定题目类型和语言
- **自动判题检测**：提交后自动检测通过/部分正确/编译错误/答案错误/格式错误/运行错误/超时/超内存等状态
- **自动修正循环**：自动模式下提交失败自动调用 AI 修正并重新提交（最多 2 次）

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 [安装脚本](https://github.com/hb-degithub/pta-/raw/main/pta-auto-solve.js) 或手动复制 `pta-auto-solve.js` 内容新建用户脚本
3. 打开 [PTA 平台](https://pintia.cn) 即可使用

## 使用

1. 进入题目页面后，脚本自动检测题目类型
2. 在悬浮面板中选择编程语言（编程题）
3. 点击「解答」获取 AI 答案，答案自动填入编辑器
4. **自动模式**：点击「全自动开始」，脚本会自动答题、提交、检测判题结果、修正错误、切换下一题
5. **手动修正**：提交报错后，在「报错反馈」输入框粘贴错误信息，点击「自动修正」获取修正后的代码

## 配置

点击悬浮面板上的「设置」按钮：

- **API URL**：AI 服务地址（支持 DeepSeek 等自动补全）
- **API Token**：你的 API 密钥
- **模型名称**：如 `deepseek-chat`
- **打字延迟**：模拟输入的延迟，0 为最快
- **自定义参数**：JSON 格式的额外请求参数

## 特性

- 绕过 PTA 粘贴限制（CodeMirror 6 编辑器直连）
- 支持多种 AI API 格式（OpenAI、DeepSeek、通义千问等）
- API URL 和模型自动匹配
- 结果独立显示区域，支持复制/填入/自动修正
- 自动确认「未保存更改」等弹窗
- 自定义请求参数（JSON 格式）

## 支持的判题状态

| 状态 | 说明 |
|---|---|
| 全部正确 | 通过检测，自动进入下一题 |
| 部分正确 | 自动调用 AI 修正 |
| 编译错误 | 自动调用 AI 修正 |
| 答案错误 | 自动调用 AI 修正 |
| 格式错误 | 自动调用 AI 修正 |
| 运行错误 / 段错误 | 自动调用 AI 修正 |
| 超时 | 自动调用 AI 修正 |
| 超内存 | 自动调用 AI 修正 |
